Project Name: Student & Admin Dashboard System

Project Overview: This is a responsive frontend-only dashboard system designed for educational management. It features two primary interfaces: an Admin Dashboard for managing student records and an Interactive Student Dashboard for viewing academic progress.  

Key Features:
    1. Admin Dashboard  
    Sidebar Navigation: Easy access to Students, Courses, Instructors, and Reports sections.  
    Dynamic Student Management: Ability to Add, Edit, and Delete student records using JavaScript DOM manipulation.  
    Real-time Statistics: Summary cards showing the total count of students, courses, and instructors.  
    LocalStorage Integration: Data persists even after the browser page is refreshed.  

    2. Student Dashboard  
    Personal Profile: Displays student name, roll number, and program details.  
    Course Summary: A detailed table listing all registered courses with faculty names.  
    Attendance System: Functional "Mark Attendance" buttons for each course.  
    Academic Records: Semester-wise result cards and fee history tables.  

Technologies Used
HTML5: Semantic structure for the dashboards.  
CSS3: Custom styling with Grid and Flexbox for responsiveness.  
JavaScript (ES6): DOM manipulation, event handling, and LocalStorage for data management.  
Google Fonts & Material Symbols: For modern typography and iconography.  

How to Run the Project
Clone the repository or download the ZIP file.
Ensure all files (AdminDashboard.html, StudentDashboard.html, Style.css, Script.js, and Logo.png) are in the same folder.  
Open AdminDashboard.html in any modern web browser to manage data.
Open StudentDashboard.html to view the student interface.
